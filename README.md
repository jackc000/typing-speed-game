# ⌨️ Typing Speed Game with Scoreboard

A fun and interactive typing game built with Python and Streamlit that helps you test your typing speed and accuracy — with a built-in leaderboard to track your best scores!

---

## 🚀 Features

- 📝 Random sentence challenges each time you play
- ⏱ Calculates typing speed (Words Per Minute)
- 🎯 Tracks typing accuracy
- 🏆 Leaderboard to display top performances
- 💾 Saves scores locally in `scoreboard.csv`
